Layer 2 — behavioral DNA and detection
Replace 5-scalar linear model with structural intelligence
v3world first
Behavioral DNA per image SHA
Now: 120-second cold-start profiling window per pod. Attacker who knows RAASA runs can wait it out.
→ Per-image-SHA Gaussian Mixture Model built from all pods of that image type. Risk = Mahalanobis distance from GMM. New pods inherit their class fingerprint at t=0. Cold-start window eliminated.
v3
LSTM temporal sequence model
Now: linear weighted sum has no memory of sequence. An attack that unfolds across 8 syscall steps over 30 seconds looks like noise.
→ LSTM on syscall 5-gram sequences. Anomaly = reconstruction error. Fast linear for triage; LSTM for confidence boosting on edge cases. Not OR — both run in parallel with different roles.
v4
Causal attribution engine
Now: the audit log says "ρ = 0.84, applied L3" — no explanation of which signal drove it or why
→ SHAP values on the risk model output. Audit log says: "escalated because syscall_distribution JSD = 0.91 (ptrace cluster, 12σ above baseline) at T+47s." Explainable AI for security.
v4world first
Game-theoretic optimal weights
Now: weights are empirically tuned (cpu=0.40, memory=0.25…). A rational attacker who reads the paper knows how to stay below the threshold.
→ Stackelberg game: defender commits to w, attacker plays best response. Solve for Nash equilibrium weight vector provably harder to evade. Publishable theorem at IEEE S&P / CCS.

Now: each node agent is fully isolated. A compromise spreading across 3 nodes looks like 3 independent events.
→ Lightweight gRPC aggregation service (or Kafka topic). Per-node agents share anomaly signals (not raw telemetry). Cluster-level risk score = weighted sum of node scores by communication graph proximity.
v4world first
Epidemiological threat propagation
Now: no predictive capability. RAASA reacts after behavior changes, not before.
→ SIR (Susceptible-Infected-Recovered) model on the pod communication graph. Compute R₀ for the cluster. When R₀ > 1, pre-emptively elevate monitoring sensitivity for high-centrality pods before they show symptoms.
v3
Behavioral fingerprint registry
Now: behavioral baselines are per-pod, rebuilt from scratch each restart. No shared knowledge across the cluster.
→ etcd-backed registry mapping image SHA → BehavioralDNA (GMM parameters). New pods inherit immediately. Old pods contribute data back. The registry improves every time a pod runs.
v4world first
Federated intelligence across clusters
Now: completely isolated. An attack pattern seen in cluster A is invisible to cluster B even if they run identical workloads.
→ Federated learning: share model parameter updates (not raw telemetry) with Gaussian differential privacy noise. Optional opt-in consortium — threat intelligence that improves across organizations.
